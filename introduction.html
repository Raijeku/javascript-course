HTML y JavaScript

En esta lección haremos una breve introducción a HTML y JavaScript.

HTML es un lenguaje de marcado que define la estructura de una página web. En el HTML se usan etiquetas que comienzan con un < y que terminan con un >. Cada etiqueta se abre y se cierra, y mientras una etiqueta se abre de la forma previamente mencionada, se cierra comenzando con un </ y terminando con un >. Las etiquetas más importantes son HTML, head, body, script y style. Dentro de la etiqueta HTML se encuentran todas las demás etiquetas. En la etiqueta head se encuentran las etiquetas que contienen metadatos (datos importantes sobre la página) y código que no debe ser visible. En la etiqueta body se encuentra todo el contenido visible de la página web. En la etiqueta script está el código JavaScript de la página y en la etiqueta style se encuentra todos los estilos y diseño de la página y los elementos de ella escritos en CSS.

Mira el código que se encuentra dentro de la etiqueta script y haz pruebas con la consola de JavaScript. Abre la consola de JavaScript con Ctrl+Shift+I.
